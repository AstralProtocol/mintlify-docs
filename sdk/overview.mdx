# SDK Overview

The Astral SDKs are the official TypeScript clients for Astral Location Services.

## Design Philosophy

- **Developer experience first**: Intuitive API that feels natural
- **Type-safe**: Full TypeScript support with comprehensive types
- **Batteries included**: Handles signing, encoding, and EAS integration

## Package Structure

```typescript
// Location attestations (separate package)
import { AstralSDK } from '@decentralized-geo/astral-sdk';

// Geospatial computations
import { AstralCompute, AstralEAS } from '@decentralized-geo/astral-compute';

// Initialize clients
const sdk = new AstralSDK({ signer: wallet });
const compute = new AstralCompute({ chainId: 84532 });
const eas = new AstralEAS(signer, 84532);

// Location attestation operations (astral-sdk)
sdk.location.create(geojson);
sdk.location.get(uid);
sdk.location.query(filters);

// Geospatial computation operations (astral-compute)
compute.distance(input1, input2, options);
compute.contains(container, geometry, options);
compute.within(geometry, target, radius, options);
compute.intersects(geom1, geom2, options);
compute.area(geometry, options);
compute.length(geometry, options);

// EAS submission helpers (astral-compute)
eas.submitDelegated(attestation);
```

## Quick Start

```typescript
import { AstralSDK } from '@decentralized-geo/astral-sdk';
import { AstralCompute, AstralEAS } from '@decentralized-geo/astral-compute';
import { ethers } from 'ethers';

// Initialize with signer
const provider = new ethers.JsonRpcProvider('https://sepolia.base.org');
const wallet = new ethers.Wallet(process.env.PRIVATE_KEY, provider);

// Initialize clients
const sdk = new AstralSDK({ signer: wallet });
const compute = new AstralCompute({ chainId: 84532 });
const eas = new AstralEAS(wallet, 84532);

// Create a location attestation
const location = await sdk.location.create({
  type: 'Point',
  coordinates: [2.2945, 48.8584]
}, {
  submitOnchain: true
});

// Compute relationship
const result = await compute.within(
  location.uid,
  landmarkUID,
  500,
  { schema: SCHEMA_UID }
);

// Submit onchain
if (result.result) {
  await eas.submitDelegated(result.delegatedAttestation);
}
```

## Key Concepts

<CardGroup cols={2}>
  <Card title="Chain Scope" icon="link">

## Configuration

```typescript
// AstralSDK configuration (for location attestations)
interface AstralSDKConfig {
  signer: Signer;               // Required: for signing attestations
}

// AstralCompute configuration (for geospatial computations)
interface AstralComputeConfig {
  chainId: number;              // Required: target chain
  apiUrl?: string;              // Optional: defaults to api.astral.global
}

// AstralEAS configuration (for onchain submission)
// Constructor: new AstralEAS(signer: Signer, chainId: number)
```
