# Changelog

## v0.1.0 — Research preview

Initial release of the Astral research preview.

- Six geocomputation operations: distance, area, length, contains, within, intersects
- TEE execution via EigenCompute (validated in test deployments; continuous attested operation is not yet funded — see [What you are trusting](/trust-model/what-you-are-trusting))
- EAS integration for onchain attestation
- TypeScript SDK
- Location proof verification (experimental; the verify service registers verifiers for several proof-of-location systems — proofmode, witnesschain, gpsd, geoclue, wifi-mls, ip-geolocation — at varying maturity, with ProofMode the most developed device-side source)
